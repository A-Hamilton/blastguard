# Code of Conduct

BlastGuard follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

Be kind. Be specific. Assume good faith. Critique the code, not the person.
English isn't everyone's first language — if a comment reads sharp, read it
again with the most charitable interpretation before responding.

Harassment, personal attacks, discriminatory remarks, and doxxing are not
welcome. Maintainers reserve the right to remove comments, close issues,
and block accounts that cross those lines.

## Reporting

If you experience or witness behaviour that violates this code, open an
issue marked private (or contact the repo owner directly via the email in
the commit log). Reports are handled confidentially.

## Scope

This applies to all BlastGuard spaces: issues, pull requests, discussions,
commits, and any adjacent venue (chat, email, conferences) where you're
representing the project.
